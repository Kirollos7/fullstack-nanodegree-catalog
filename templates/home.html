Welcome Miguel's catalog of products. :)

This project is a working example of a Flask + Python 3 application which manages a catalog of products. The persistence layer of the application uses PostgreSQL.

Additionally, the application:

  • Integrates with Github OAuth to manage authentication of users (try to login!)
  • Allows authenticated users to create and manage their own products
  • Places the Github profile avatar of the author next to the product
  • Runs with configuration from configuration files which are only available in the server
  • Runs in an Amazon Web Services (AWS EC2) instance in the cloud, via a Apache2 + WSGI application
I setup and configured the entire stack, providing a web front-end, simple REST APIs, Database setup and initial data, Ubuntu Linux setup on AWS EC2 ins tance.

You may find the source code, please post issues, and fork my project in: https://github.com/miguelrincon/fullstack-nanodegree-catalog
